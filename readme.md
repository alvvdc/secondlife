## Docker:

#### Iniciar / Parar:
```bash
docker-compose start
docker-compose stop
```

#### Borrar:
```bash
docker-compose down
```

#### Limpiar cache:
Si realizamos ajustes en la configuración de docker-compose y no se aplican deberemos limpiar el cache con el siguiente comando.
```bash
docker system prune -a
```
